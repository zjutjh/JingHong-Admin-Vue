# 微精弘管理后台

使用 Vue3 + TS + Native UI 开发的Web网站

## 介绍
微精弘小程序的管理后台

## 技术栈
- Vue3

- Typescript

- Pinia

- Axios

## 快速开始

1. fork仓库并且clone到本地

2. 在根目录依照`.env.sample`配置`.env`文件
  
3. 安装依赖 `pnpm i`
  
4. 运行`pnpm run dev`

5. 点击本机地址预览

6. Coding


## 打包部署

- `pnpm run build` 后将生成在根目录的dist文件夹压缩为压缩包发给后端小伙伴
